# Список задач

Менеджер задач с категориями и сохранением данных между сессиями.

## Что умеет

- Добавлять задачи с выбором категории: Работа, Личное, Учёба
- Отмечать задачи выполненными
- Редактировать текст задачи
- Удалять задачи
- Фильтровать задачи по категории
- Сохранять всё в localStorage - данные не пропадут при перезагрузке

## Технологии

HTML, CSS, JavaScript (без библиотек), localStorage

## Запуск

Открыть `index.html` в любом браузере.
